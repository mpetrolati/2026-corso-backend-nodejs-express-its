# 01 — Hello Node

Il primissimo script Node.js. Mostra:

- come eseguire un file `.js` con Node
- le variabili globali di Node (`__filename`, `__dirname`, `process`)
- la differenza con il JavaScript del browser (qui non c'è `window`, `document`, ecc.)
- come leggere argomenti dalla riga di comando

## Esecuzione

```bash
node hello.js
node hello.js Mario
```
